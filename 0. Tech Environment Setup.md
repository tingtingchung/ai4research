The following tools are used in this repo. Follow instructions to install/purchase.

1. To run Claude Code, you need to have
   - A paid Claude Pro ($20/month) account on [Claude](https://claude.ai/new)
   - A desktop work environment - Choose one of these two options:  
     A. [Claude Desktop](https://claude.com/download?utm_source=google&utm_medium=pse_knwl&utm_campaign=acq_cowo_us&utm_content=use_txt_v1&utm_term=brandcowork&gclsrc=aw.ds&gad_source=1&gad_campaignid=23532527189&gbraid=0AAAAA99jmqsPSKg03FLRk1T9wywpa4crh&gclid=Cj0KCQiA7-rMBhCFARIsAKnLKtCFPkBjTV0Op9IRiAeAu0vkwAeptL5JUE2tSnVObIVjmvcM3aioT-8aAtXaEALw_wcB)  
     B. Claude Code in VS Code
     - [VSCode](https://code.visualstudio.com/): Download and install for free
     - [Install Claude Code in VSCode](https://www.youtube.com/watch?v=ph5DRCX_g6s): Follow instructions in this 3-minute video.
   - [Python](https://www.python.org/downloads/): Download and install for free - Be sure to select "Add Python to your path".
   - [pip](https://youtu.be/q2DbpErvXGQ): Follow instructions here to install pip in VSCode.
   - Tip: Once you have VSCode and Claude Code, you can ask Claude to install Python and pip for you!
